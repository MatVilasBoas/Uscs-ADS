Questões: 1> A linguagem HTM surgiu com a ideia de criar um sistema de hipertexto(links) na internet, ele é utilizado para construir páginas na web e atualiza-las, as características são as marcações(tags), todos os elementos(texto, imagem, vídeo e áudio) são definidos pelos caracteres entre as tags "<>". 2> O software precisa ter uma conexão com um navegador, o principal software é o Visual Studio Code. 3> é o cabeçalho, contém as definições que o navegador deverá seguir para renderizar a página web. é tudo que será exibido para os usuários (parágrafos, listas, imagens, links, animações...etc).
