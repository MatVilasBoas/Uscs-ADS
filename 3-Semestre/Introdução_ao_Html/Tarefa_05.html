A Linguagem de Programação é uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas de software.

Reforçando esse conceito, Silas dos Santos Carneiro define:

"Entende-se por linguagem de programçao ao conjunto de regras sintáticas definidas pelo usuário e que devem ser executadas pelo computador para a realização de tarefas, por exemplo, a soluçao numérica de problemas matemáticos ou aquisição de medidas de experimentos em laboratório."
