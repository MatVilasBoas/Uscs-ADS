Exemplo de Lista de Descrição

TCP
O protocolo de Controle de Transmissão (TCP) é um método (protocolo) usado junto com o Protocolo da Internet (IP) para enviar dados na forma de unidades de mensagem, chamadas de pacotes, entre computadores pela Internet.
IP
O protocolo da Internet (IP) é o método ou protocolo pelo qual os dados são enviados de um computador para o outro na Internet. Cada computador na internet é identificado de forma única por um endereço IP.
HTTP
O protocolo de Tranferência de Hipertexto (HTTP) é o protocolo usado para trocar texto, imagens gráficas, som, vídeo e outros arquivos multimídia na Web.
